# Dades

Les dades de població d'aquest projecte es corresponen amb les dades del padró municipal espanyol de l'any 2024, publicades per l'Instituto Nacional de Estadística (INE) sota llicència CC BY 4.0 [https://ine.es/dynt3/inebase/es/index.htm?padre=525]

Es corresponen amb una sèrie històrica referenciada a data 1 de gener que comença l'any 1996

Les dades geogràfiques d'aquest projecte han estat obtingudes del Centro de Descargas del Instituto Geográfico Nacional, també sota llicència CC BY 4.0 


# Data

The population data for this project correspond to the data from the Spanish municipal register for the year 2024, published by the Instituto Nacional de Estadística (INE) under a CC BY 4.0 license [https://ine.es/dynt3/inebase/es/index.htm?padre=525].

They correspond to a historical series referenced on January 1st, starting in 1996.

The geographic data for this project were obtained from the Centro de Descargas del Instituto Geográfico Nacional, also under a CC BY 4.0 license
