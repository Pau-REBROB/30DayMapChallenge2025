# Dades
Les dades d'aquest projecte es corresponen amb el dataset "Habitatges d'ús turístic de la ciutat de Barcelona", propietat de l'Ajuntament de Barcelona i distribuïdes sota llicència CC BY 4.0 per Open Data BCN, el Servei de Dades Obertes de l'Ajuntament de Barcelona.

Van ser publicades per primer cop el 27/07/2016, i s'actualitzen setmanalment.


# Data
Data for this project corresponds to the dataset "Tourist housing in the city of Barcelona", owned by the Barcelona City Hall and distributed under a CC BY 4.0 license by Open Data BCN, Barcelona's City Hall Open Data Service. 

They were first published on 27/07/2016 and are updated on a weekly basis.
